# Defontana Challenge

## Instalación y ejecución ⚙️

- Crear archivo .env

  ```bash
  VITE_POKEAPI_BASE_URL="https://pokeapi.co/api/v2"
  ```

- Instalar dependencias y levantar proyecto

  ```bash
  npm i

  npm run dev
  ```
